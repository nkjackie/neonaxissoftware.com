# NeonAxis Software Website

This repository contains the public marketing website for **NeonAxis Software LLC**, a U.S.-registered technology company.

NeonAxis focuses on:

- Software infrastructure and cloud-native services  
- Data & analytics engineering  
- AI-assisted automation, modeling, and decision-support tools  

The site is built as a static, multi-page website and deployed via **Cloudflare Pages**.  
There are no financial services, payments, or custodial products provided by this codebase; it is strictly a software and information website.

## Tech stack

- Static HTML, CSS, and minimal JavaScript  
- Deployed on Cloudflare Pages  
- Optional Cloudflare Worker for redirects (e.g. `www` → apex domain)

## Structure

- `index.html` – Home  
- `products.html` – AI & infrastructure products  
- `about.html` – Company & team overview  
- `contact.html` – Contact details
